# Money Stories — Channel Setup Guide

Step-by-step to create and configure the YouTube channel. Work top to bottom;
the order matters (account type and phone verification unlock later steps).

## 0. The Google account (hardest to undo — do first)

- Use a dedicated Google account (e.g. `moneystories.channel@gmail.com`) or your
  existing account **plus a Brand Account channel** (step 1). Never build directly
  on a personal identity.
- Enable **2-step verification** immediately. Finance channels are hijack targets.
- Add recovery email + phone.

## 1. Create the channel as a Brand Account

youtube.com → profile picture → **Settings** → **Create a channel** →
**"Use a custom name"** → **Money Stories**.

The "custom name" path creates a Brand Account: renameable, supports multiple
managers, separate from your personal Google profile.

## 2. Handle

Studio → **Customization → Basic info → Handle**. Preference order:

1. `@moneystories` (likely taken)
2. `@MoneyStoriesShow`
3. `@MoneyStoriesTV`
4. `@TheMoneyLedger`
5. `@MoneyStoriesArchive`

Register the same handle on Instagram / TikTok / X the same day.

## 5. Phone verification

studio.youtube.com → **Settings → Channel → Feature eligibility** → verify by phone.
Unlocks **custom thumbnails**, **videos >15 min**, appeals. Required before first upload.

## 6. Channel-wide settings (Studio → Settings)

- **Channel → Keywords:** `money stories, personal finance, compound interest,
  investing, quiet millionaire, financial education, documentary`
- **Channel → Advanced → Audience:** "No, it's not made for kids" (channel level)
- **Upload defaults → Basic info:** paste the footer from `DESCRIPTION-TEMPLATE.md`;
  default tags; category **Education**

## 9. Launch sequence (per plan §8 — never launch with one video)

1. Produce Episodes 1–3 fully.
2. Upload Ep 1 **unlisted** → QA end-to-end on phone + TV (audio, captions, chapters).
3. Set channel trailer = Ep 1 cold open (Customization → Layout).
4. First Short 2 days before launch; publish Ep 1 Saturday ~10am ET;
   schedule Eps 2–3 for the following Saturdays.

## 10. Day-one hygiene

- Comments: **Hold potentially inappropriate comments for review**; blocked words:
  `WhatsApp`, `telegram`, `+1(` (standard impersonation-scam patterns).
- Monday 30-min analytics review (plan §9 KPIs: hook retention >70%, AVD >40%, CTR 4–6%).
- Monetization (YPP) is a later milestone: 1,000 subs + 4,000 watch hours — nothing
  to configure now beyond staying advertiser-friendly and disclosed.
